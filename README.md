\# Zynq SoC Accelerator



\## Overview

This project implements a Zynq-based SoC accelerator design using Verilog HDL, Vivado, and Vitis.



The system integrates a hardware FIR accelerator with Zynq processing system support and FPGA-ready design files.



\## Features

\- Zynq SoC accelerator architecture

\- FIR hardware accelerator

\- AXI/FPGA-ready integration structure

\- RTL source files

\- Testbench and simulation files

\- Constraint files

\- Vitis/Linux application support

\- Generated XSA and bitstream proof included



\## Tools Used

\- Verilog HDL

\- Vivado

\- Vitis

\- ModelSim
